# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Instructions

This project uses comprehensive instruction files for AI agents. Read these before starting work:

### Primary Instructions

1. **`.github/copilot-instructions.md`** - Generic AI agent guidelines (coding patterns, workflows, terminal commands)
2. **`docs/requirements.md`** - Project requirements, domain model, organization
3. **`docs/conventions.md`** - Coding standards, CI/CD, versioning
4. **`docs/agents.md`** - Agent-based workflow overview

### Architecture Documentation

- **`docs/adr-*.md`** - Architecture decision records (created per feature as needed)

### Development Guidelines

- **`docs/features.md`** - User-facing feature documentation
- **`CONTRIBUTING.md`** - Contribution guidelines, branch strategy, commit conventions
- **`README.md`** - Project overview and usage

### Agent Skills

Complex workflows are documented in **`.github/skills/<skill-name>/SKILL.md`**. Check for relevant skills before starting work.

### Specialized Agents

Agent-specific instructions are in **`.github/agents/<agent-name>.agent.md`**.

### Prompts

Reusable prompts are in **`.github/prompts/<prompt-name>.prompt.md`**.
