# Tasks: GreenLedger MVP

## Overview

This document breaks down the GreenLedger MVP feature into 27 implementation tasks across
six phases. The MVP delivers a full-stack Next.js application for CSRD/ESRS E1 climate
reporting: supplier management, Scope 1/2/3 emissions recording, proxy calculations,
methodology documentation, and PDF export.

**References:**

- Specification: [`docs/features/001-mvp/specification.md`](./specification.md)
- Architecture: [`docs/features/001-mvp/architecture.md`](./architecture.md)
- Test Plan: [`docs/features/001-mvp/test-plan.md`](./test-plan.md)

**Acceptance Criteria:**
